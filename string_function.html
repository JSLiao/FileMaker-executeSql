string function



Left 相當於 FileMaker 的 feft function

Left ( 字串, 字元數)

Right 相當於 FileMaker 的 right function

Right ( 字串, 字元數)

SubStr, SubString 接近於 FileMaker 的 middle function

如果字元數沒寫, 則從起始位置取到字串結束

SubStr ( 字串, 起始位置[, 字元數])

Left, Right, SubStr, SubString 不能和 Distinct 搭配使用

Lower 把字串的所有字元轉成小寫

Lower ( 字串 )

Upper 把字串的所有字元轉成大寫

Upper ( 字串 )

LTrim 刪除字串開頭的空白

LTrim (字串)

RTrim 刪除字串結尾的空白

RTrim (字串)

Trim 刪除字串開頭和結尾的空白

Trim (字串)

LTrim, RTrim, Trim 可以和 Distinct 搭配使用

Space 產生指定個數空排字元的字串

Space (空白字元個數)

Length 取得字串長度

Length ( 字串)
