Select clause



column 必須是在 From 子句中 table 中的欄位或者是在 Where 子句或 Join 子句有建立關係的, 是必須的子句

Select 子句的 column 和 其他敘述是不分大小寫

其書寫的順序就是取值結果的順序

Select 子句中如果有聚合函數, 則不在聚合函數內的 column, 必須在 Group By 子句中有列出

column 的書寫方式: 純欄位或者 baseTableName.fieldName

可以使用別名 ( alias ) 讓敘述更語意化, 或者縮短敘述

可以使用運算子改變顯示的樣式

可以使用子查詢( sub-query)

可以使用函數

可以使用 argument

* 表示取得所有 column ( 如果使用 * ,  Select 子句不可以其他敘述)

比較運算子不可以用在 Select 子句

column 名稱的開頭第一個字元如果不是字母字元, 則column 名稱需要用雙括號括起來
