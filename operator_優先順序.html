operator 優先順序



在同一列載運算式中的優先順序, 依照運算式從左至右

順序運算子
1( ) 括號
2'' 單引號
3Unary '-', Unary '+' ( 數字的正負符號)
4^, **
5*, /
6+, -
7|| 串接運算子
8 =, <>, <, <=, >, >=, Like, Not Like, Is Null, Is Not Null, Between, In, Exists, Any, All
9Not
10AND
11OR

計算式:

/*
找出台北市和台中市的有限公司, 但不是股份有限公司
*/
executeSQL("
  Select name, addrCity 
  From company 
  Where name Like '%有限公司%' And 
    name Not Like '%股份%' And 
    addrCity In ('台北市', '台中市' )
  Fetch First 10 Rows Only
  "; "" ; "" )
