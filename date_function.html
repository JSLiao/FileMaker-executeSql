date function



Day: 

取得日期的日的部分, 取得的值是數字類型資料

Day ( Date ‘yyyy,mm,dd' )

DayName: 

取得日期是星期幾的英文名, 取得的值是字串類型資料

DayName( Date ‘yyyy,mm,dd' )

DayOfWeek: 

取得日期是星期幾的數字表示, 取得的值是數字類型資料, 1 表示星期日,..., 6 表示星期六, 和 FileMaker 相同

DayOfWeek ( Date ‘yyyy,mm,dd' )

Month: 

取得日期的月的部分, 取得的值是數字類型資料

Month( Date ‘yyyy,mm,dd' )

MonthName: 

取得日期的月的英文名, 取得的值是字串類型資料

MonthName ( Date ‘yyyy,mm,dd' )

Year: 

取得日期的年的部分, 取得的值是數字類型資料

Year ( Date ‘yyyy,mm,dd' )

Date: 

表示日期, 使用四位數表示, 不要使用二位數的年

Date ‘yyyy,mm,dd'

合法日期: 從date ‘0001-01-01’  到 date ‘4000-12-31'

在 FileMaker Date ( 5 ; 32 ; 2022 )  會被自動換算成  2022/06/01

在 Sql Date ‘2022-05/32’ 則會被認定為錯誤的數據輸入

日期可以對日期做相加減到運算, 得到的數字資料

日期可以對數字做相加減運算

如果所得的日期是在合法日期的範圍內: 結果為日期

如果所得的日期是早於 date'0001/01/01’, 則得到數字資料

如果所得的日期是晚於 date’4000/12/31’, 則發生錯誤

這和 FileMaker 相同


計算式:

executeSql ( "Select Day( Date'2022-5-22'),DayOfWeek(Date'2022-5-22'),DayName ( Date'2022-5-22') From oneRecord "; ", " ; "" ) & ¶ & 
executeSql ( "Select Month( Date'2022-5-22'),monthName ( Date '2022-5-22') From oneRecord "; ", " ; "" ) & ¶ &
executeSql ( "Select Year( Date'2022-5-22') From oneRecord "; ", " ; "" )

計算結果:

22, 1, Sunday
5, May
