mathematical functions



數值函數 (SQL Mathematical Functions)

Abs 函數用來取得絕對值。

Abs (數值)

Ceil、Ceiling 函數取得不小於指定數值的最小整數。

Ceil(數值)

Floor 函數取得不大於指定數值的最大整數。

Floor(數值)

Sqrt 函數取得數值的平方根。

Sqrt (數值)

Log 函數取得以 10 為底數的對數。

Log(數值)

Ln 函數取得以 e為底數的對數。

Ln (數值)

Exp 函數取得以 e為底數的指數。 e 是數學常數, 也就是自然對數函數的底數，近似值為 2.718281828, 又稱為歐拉數。

Exp (數值)

Power 函數取得數值的 N 次方值。 FileMaker 不支援 Power 的寫法; 可以使用運算子"^" 或者 "**"。

數值 ^ 次方 

或者

數值 ** 次方

Mod 函數取得數值被除數對除數的餘數。

Mod ( 被除數 , 除數)

Int 函數取得數值的整數部分。

Int ( 數值)

Sign 函數取得數值的正負數符號。如果是正數取得 +1 ; 如果是負數取得 -1。

Sign ( 數值)

Round 函數取得指定小數位數的四捨五入數值, 小數位數可以是負數, 同 FileMaker。

Round ( 數值,  小數位數)

Pi 圓周率

PI()


計算式:

// Abs 函數用來取得絕對值。
ExecuteSQL ( "Select abs( -11.12 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Ceil、Ceiling 函數取得不小於指定數值的最小整數。
ExecuteSQL ( "Select ceil( 11.12 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select ceil( -11.12 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Floor 函數取得不大於指定數值的最大整數。
ExecuteSQL ( "Select floor( 11.12 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select floor( -11.12 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Sqrt 函數取得數值的平方根。
ExecuteSQL ( "Select Sqrt( 16 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Log 函數取得以 10 為底數的對數。
ExecuteSQL ( "Select log( 16 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Ln 函數取得以 e為底數的對數。
ExecuteSQL ( "Select ln( 3 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Exp 函數取得以 e為底數的指數。 e 是數學常數, 也就是自然對數函數的底數，近似值為 2.718281828, 又稱為歐拉數。
ExecuteSQL ( "Select exp( 3 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Power 函數取得數值的 N 次方值。 FileMaker 不支援 Power 的寫法; 可以使用運算子"^" 或者 "**"。
ExecuteSQL ( "Select 2^3 From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select 2**3 From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select 2** ( 1/3) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Mod 函數取得數值被除數對除數的餘數。
ExecuteSQL ( "Select mod( 9,4 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select mod( 9,-4 ) From oneRecord" ; "";"" ) & ¶ & 
ExecuteSQL ( "Select mod( -9,4 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select mod( -9,-4 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Int 函數取得數值的整數部分。
ExecuteSQL ( "Select int( 16.12 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select int( -16.12 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Sign 函數取得數值的正負數符號。如果是正數取得 +1 ; 如果是負數取得 -1。
ExecuteSQL ( "Select sign( 16.12 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select sign( -16.12 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select sign( 0 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Round 函數取得指定小數位數的四捨五入數值, 小數位數可以是負數, 同 FileMaker。
ExecuteSQL ( "Select round( 12.355, 2 ) From oneRecord" ; "";"" ) & ¶ &
ExecuteSQL ( "Select round( -12.355, 2 ) From oneRecord" ; "";"" ) & ¶ & ¶ &

// Pi 圓周率
ExecuteSQL ( "Select pi() From oneRecord" ; "";"" )
