concat 串連運算子 || 和字串連接運算子 +



字串連接運算子 +

只能串接字串和字串欄位

合併字串欄位

(取代 executeSql 指定的欄位分隔符號)

concat 串連運算子 || 

串接字串和欄位

合併欄位, 但不能合併數字欄位與數字欄位

(取代 executeSql 指定的欄位分隔符號)

計算式:

// 使用 column separator
executeSql("
  Select '中文名',name, '英文名', nameEng 
  From contact 
  Where addrCity = ? 
  Fetch First 5 Rows Only
  "; ": " ; "" ; "基隆市")

/*
使用 "+" 串接字串和字串欄位
*/
ExecuteSQL("
  Select 'id: ' + id + '中文名: ' + name + ' 英文名: '+ nameEng
  From contact 
  Where addrCity = ? 
  Fetch First 5 Rows Only"; "===" ; "" ; "基隆市")

計算式:

/*
使用 "||" 串接字串和數字欄位
取得預期的結果: 
*/
ExecuteSQL("
  Select '中文名: '|| name || ' 英文名: '|| nameEng || ' 年齡: ' || age 
  From contact 
  Where addrCity = ? 
  Fetch First 5 Rows Only"; "===" ; "" ; "基隆市")
