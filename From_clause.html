From clause



From 子句用來指定在 Select 子句中 column 的 base table name, 是必須的子句

From 子句最少指定一個 table, 有時因需要而列出兩個以上的 table

From 子句列出的 table 愈多表示必較耗用記憶體, 計算比較費時

可以用 Join 子句或者 Where 子句來建立與主要查詢的 table ( left table ) 的關係,  通常使用 Join 子句可以獲得必較好的效率

Join 子句中指定的  right table 不能出現在 From 子句

計算式:

/*
From sellItem, sell, employee
漏掉任何一個 table 都會發生錯誤
花費時間: 2.678 sec
*/
executeSql ( "
  Select 
    (Select employee.name from employee where employee.id = sell.salesRep),
    sum(sellItem.item_qty * sellItem.item_unitPrice )
  From sellItem, sell, employee
  Where sell.id = sellItem.idf and employee.id = sell.salesRep
  Group by sell.salesRep
  "; "" ; "" )

/*
join 子句中已經指定的 table 不能再放在 From 子句
這個寫法, 效率很差
花費時間: 32.678 sec
*/
ExecuteSQL ( "
  Select 
    (Select employee.name from employee where employee.id = sell.salesRep),
    sum(sellItem.item_qty * sellItem.item_unitPrice )
  From sellItem, employee
  Join sell on sell.id = sellItem.idf
  Where sell.id = sellItem.idf and employee.id = sell.salesRep
  Group by sell.salesRep
  "; "" ; "" )
