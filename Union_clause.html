Union clause



Union 運算子用來將兩個(以上) SQL 查詢的結果合併起來，而由 Union 查詢中各別 SQL 語句所產生的欄位需要是相同的資料型別及順序。

Union 查詢只會返回不同值的資料列，有如 Select Distinct。

Union All 則會列出所有的資料，不論是否重複。

Union 就是像是 OR (聯集)，如果紀錄存在於第一個查詢結果集或第二個查詢結果集中，就會被取出。

Union 與 Join不同的地方在於，Join 是作橫向結合 (合併多個資料表的各欄位)；而 Union 則是作垂直結合 (合併多個資料表中的紀錄)。

可以加上 Offset 和 Fetch First, 但只能對合併後的資料做處理

計算式:

executeSql("
  SELECT contact.name, contact.addrCity, contact.addrStr 
  FROM Contact 
  WHERE addrCity = '彰化縣' and contact.name Like '%日%'
  UNION
  SELECT contact.name, contact.addrCity, contact.addrStr 
  FROM Contact 
  WHERE addrCity = '嘉義縣' and contact.name Like '%日%'
  "; "" ; "")

計算式:

executeSql("
  SELECT contact.name, contact.addrCity, contact.addrStr FROM Contact WHERE addrCity = '彰化縣'
  UNION
  SELECT contact.name, contact.addrCity, contact.addrStr FROM Contact WHERE addrCity = '嘉義縣'
  Offset 150 Rows
  Fetch First 10 Rows Only
"; "" ; "")
/*
雖然可以加上 Offset 和 Fetch First, 但只能對合併後的資料做處理
