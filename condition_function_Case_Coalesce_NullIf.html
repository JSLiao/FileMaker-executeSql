condition function, Case, Coalesce, NullIf



Case 

類似於程式語言裡的 if/then/else 語句，用來作邏輯判斷。

CASE
  WHEN condition THEN result
  [WHEN···]
  [ELSE result]
END;

或是：

CASE expression
  WHEN value THEN result
  [WHEN···]
  [ELSE result]
END;

若省略了 ELSE 子句且洽無相符的條件則會返回 NULL。

Coalesce 

取得第一個運算式為不是NULL 的目前值。

Coalesce 可以視為 Case 函數的一種精簡敘述

expression 的 data type 要相同

COALESCE ( expression1[,expression2... ] )

NullIf 

如果兩個運算式不相等，NullIf 會傳回第一個 expression。 

如果運算式相等，NullIf 會傳回 Null 。

NULLIF ( expression , expression )

/* 
ageRange 是隨意取的名稱, 可以不寫 As ageRange
*/
executeSql("
  SELECT name,nameEng,
    CASE 
      WHEN age <= 18 THEN 'teen'
      WHEN age <= 30 THEN 'young'
      WHEN age <= 50 THEN 'middle age'
      WHEN age <= 100 THEN 'old'
      ELSE 'monster'
    END AS ageRange
  FROM contact 
  WHERE addrCity = ? FETCH FIRST 10 ROWS ONLY
  "; "  -  " ; "" ; "台北市")
