Distinct



一個資料表的某欄位中可能會有多個紀錄都是相同值的情況，在 Select 查詢語句中我們可使用 Distinct 關鍵字過濾重複出現的紀錄值。

Select Distinct table_column1, table_column2...
From table_name;

若 Distinct 後面有兩個以上的欄位，則要所有的欄位的值都相同才會視為重覆, 在顯示內容中被剔除。

只有其中一個欄位值不同，仍會顯示該筆資料。

Distinct 後面有多個欄位時, 效率會降低。

Distinct 的輸出結果會依據 ASCII code 排序

Distinct 是區分大小寫的

如果要取得不區分的輸出結果

Distinct  Lower (  fieldName ) 或 Distinct  Upper (  fieldName ) 

如果 Distinct 配合使用 Lower 或 Upper 函數, 則輸出的結果會變成小寫或大寫

Distinct 雖然可以和 Group By 一起使用, 但必須: 在 Select 子句中出現的欄位都必須在 Group By 語句中, 這沒有實際意義, 且降低效率裝框非常嚴重, 不要這樣使用

計算式:

/* 沒有用 Distinct, 重複的值會被列出
executeSql ( "Select addrCity From company " ; "" ; "" )
*/
// Distinct: 重複的值會被省略
executeSql("
Select Distinct addrCity 
From company
" ; "" ; "" )
